# 서울시의회 회의록 시민 서비스 (프로토타입)

시민이 쉽게 이해할 수 있는 서울시의회 회의록 UI 프로토타입입니다.

## ✨ 주요 기능

### 1. 📊 요약 보기
- **한 눈에 파악**: 회의 핵심 정보를 3초 안에 확인
- **주요 결정사항**: 안건별 쉬운 설명 제공
- **시각적 타임라인**: 회의 진행 과정을 한눈에

### 2. 🔍 자세히 보기
- **안건 카드**: 각 안건의 상세 정보
- **쉬운 설명**: 일반 시민도 이해하기 쉬운 언어로 번역
- **첨부문서 링크**: 관련 자료 다운로드
- **참석자 정보**: 발언자 목록 및 프로필

### 3. 📝 전문 보기
- **전체 회의록**: 원문 그대로 보기
- **발언자 필터**: 특정 의원의 발언만 선택적으로 보기
- **검색 기능**: 키워드로 발언 내용 검색

### 4. 📖 용어 사전
- **의정 용어 설명**: 어려운 행정/법률 용어를 쉽게 설명

## 🚀 실행 방법

브라우저에서 **http://localhost:3000** 접속

현재 개발 서버가 실행 중입니다!

## 🎯 향후 개발 계획

- LLM 통합 (자동 요약, 쉬운 말 변환)
- 챗봇 기능 (RAG)
- 안건 추적 시스템
- VectorDB 검색

## 🛠️ 기술 스택

- Frontend: Next.js 16 + TypeScript + Tailwind CSS
